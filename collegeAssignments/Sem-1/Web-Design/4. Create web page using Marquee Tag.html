Marquee Tag and its Attributes

Marquee

The Marquee tag in HTML is used to create a scrolling or moving text or image element on a web page

Atttributes of Marquee tag

1. Behavior

This attribute determines the scrolling behavior,it can have one of the following values

I.Scroll

The content continuously scrolls in the specified direction

II.Alternate

The content scrolls back and forth, reversing direction when it reaches the end.

III.Slide

The content slides into view from one direction and then slides out in the opposite direction.

2. direction

This attribute sets the direction of scrolling, it can have one of the following values

I.Left

The content continuously scrolls in the specified direction

II.Right

The content scrolls back and forth, reversing direction when it reaches the end.

III.Up

The content slides into view from one direction and then slides out in the opposite direction.

IV.Right

The content scrolls back and forth, reversing direction when it reaches the end.

3. scrollamount

his attribute specifies the speed of scrolling

4. scrolldelay

this attribute specifies the speed of scrolling

5. width and height

These attributes define the dimensions of the marquee element.

6. Loops

The loop attribute indicates how many times the marquee should repeat its scrolling behavior.
